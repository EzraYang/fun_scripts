### 功能
* 把kindle自动生成的'My Clippings.txt'里同一本书的笔记提取出来。
* 加入‘根据位置排序’功能。  
Kindle默认排序是按照标注时间。例如第一次阅读某书，在位置#100， #200， #300标注；第二次读在#150，#250标注。导出笔记的默认排序是#100，#200，#300，#150，#250。而更合理的排序是按照位置来排。
